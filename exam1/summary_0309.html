이벤트 선언

$('.box').on('click', function () {
    $('.box').css({
        'border' : '1px solid red'
        'background' : 'blue'
    })
})
$('.box').on('click', changCSS);

함수선언과 호출

1) 선언함수

changeCSS();
function changeCSS() {
    $('.box').css({
        'border' : '1px solid red'
        'background' : 'blue'
    })
}
  • 이슈 : 함수호출구문이 함수선언구문 앞에 와도 작동하여 순서혼란 가능. 가급적 사용 지양

2)익명함수

var chageCSS = function () {
    $('.box').css({
        'border' : '1px solid red'
        'background' : 'blue'
    })
}
changeCSS();
  • 호출구문이 반드시 함수선언 뒤에 와야함
