Study Summary - #02

window.onload vs $(document).ready

  • window.onload : DOM + 모든 리소스를 로드한 후 호출
  • $(document).ready : DOM을 로드한 후 호출. javascript에서는 미지원. 적용하려면 긴 코드가 필요

변수

  • 변수 : 변동하는 값을 일정 변수로 치환 가능
  • 변수 적용하기 tip : 우선 각 요소를 변수없이 직접 입력해 본 후, 동일하게 반복되는 값이나 규칙성 있게 변화하는 값을 찾아 변수로 치환
