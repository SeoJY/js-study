Study Summary - #11 Sticky Menu

Tip

  • filter로 id를 가진 a만 가져오기
    • filter() : 앞의 셀렉터로 선택된 요소 중에서 true인 것만 가져옴. for문을 굳이 사용하지 않아도 됨
  • 컨텐츠의 top 위치 구하기 : .offset().top; 
    $('html, body').scrollTop(top값);
$('html, body').animate({scrollTop : target})
  • 각 영역의 offset.top()을 배열에 담아 활용
  • 그 배열에서 클릭한 타겟 부모의 인덱스를 가져옴
  • 스크롤할 때 scrollTop값을 가져와서 배열의 top값과 비교하여 index값을 반환
  • 해당 index값으로 앵커목록에 활성화 class넣었다 빼기
  • 활성화class가 이미 들어가있는 경우에는 return반환
  • 스크롤 이벤트를 걸었을 경우, 스크롤이 내려간 상태에서 페이지 새로고침을 했을 경우, 해당 함수가 실행되지 않음
    => 처음 로드시 스크롤 이벤트에 걸린 함수를 한번 실행해준다
  • 스크롤도중에 함수가 계속 실행되는 이슈
    -> 스크롤이 끝난 시점에서만 실행되게 만들기 
    win.clearTimeout(this.scrollTimeOut);
this.scrollTimeOut = win.setTimeout($.proxy(this.activeClassFunc, this), 100)
