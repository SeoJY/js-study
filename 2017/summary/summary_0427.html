유용한 플러그인(telltrue33.github.io)

  • convention : 하이브랩 컨벤션
  • example
    • animate-enhanced
      • 단점 : 플러그인이 한번만 동작. 여러번 발생하지 않음 -> enhanced를 사용하지 않는 animate 함수 CSS 속성에 avoidCSSTransition: true를 추가
      • ex) nike 광고
    • domchange : dom구조가 바뀌는 것을 detecting
      • prepend, append, before, after, html
      • innerHtml은 detecting불가
    • slide
      • slick
        • 장점 : fade 메서드 지원
        • 터치의 가속도와 상관없이 1개씩 움직임
      • swiper2
        • 부드러운 터치 슬라이드를 구현할 수 있는 플러그인
        • 단점 : fade 메서드가 없다
      • swiper3
        • 부드러운 터치 슬라이드를 구현할 수 있는 플러그인
        • IE10+
    • trigger : event를 custom
    • image loaded : 이미지 로딩되는 시점 체크
    • video : video 태그를 지원하지 않는 브라우저 호환
